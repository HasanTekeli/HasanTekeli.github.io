ALMS2OBS - The very first project on Github!

This project is created using Flask to make things easier for instructors. If you are using AkademikLMS (as LMS) and OBS (Proliz) (as SIS), you can use this project to easily modify Excel files (results of the exams) downloaded from ALMS to paste the grades of the students to OBS.

This project grabs the folder of the downloaded Excel files and does a few things for you.

Things done in the backend:

  • Renames the files according to the C2 cell of the files.
  • Trims the unnecessary columns and leaves the columns of only students' numbers and grades.
  • Orders the grades from the highest to lowest.
  • Saves the files to the newly created mods folder inside of the chosen folder.

Things done in the frontend:

  • Lists all the files in the mods folder.
  • Copies the numbers and grades of the students to the clipboard when clicked on one of the department names.
  • While copying, it excludes the students' numbers with a grade of "-1"(meaning the student hasn't attended the exam)
